# Development Roadmap

## Phase 0: Setup (Day 1-2)

- Cloudflare accounts, D1/R2/KV setup.
- Enable Browser Rendering API.
- Repo scaffolding.

## Phase 1: Core MVP (Week 1)

- Worker gateway with validation, cache, storage.
- Browser Rendering + AI pipeline.
- Basic frontend form + report page.

## Phase 2: Production Hardening (Week 2)

- Rate limits, retries, timeouts.
- Observability, logging, error tracking.
- SEO basics and sitemap.

## Phase 3: Growth (Week 3-4)

- Blog + programmatic SEO pages.
- Pricing page and upgrade CTA.
- Performance optimizations.

## Phase 4: Monetization (Optional)

- Stripe checkout and plan limits.
- Extended report retention.
