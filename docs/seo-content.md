# SEO & Content Plan

## SEO Goals

- Rank for "above the fold analyzer", "landing page CRO audit", "landing page critique".
- Use programmatic SEO for industries and use cases.
- Emphasize "free, no login" in titles and meta where relevant.

## Technical SEO

- SSR with clean HTML output.
- XML sitemap auto-generated.
- Robots.txt allow indexing of public reports and blog.
- Canonical tags for report pages.

## Structured Data

- Product schema on landing.
- FAQ schema on pricing.
- Article schema on blog posts.
- Review/Rating schema on report pages (if allowed).

## Content Architecture

- Core pages: /, /analyze, /pricing, /blog
- Programmatic pages: /industry/[slug] (e.g., /industry/saas)
- Programmatic use cases: /use-case/[slug]
- Aggregations: /best-landing-pages, /common-mistakes
